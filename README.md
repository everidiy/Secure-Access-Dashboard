# Secure-Access-Dashboard

**Modern admin dashboard for managing users** built with **React, TypeScript, Zustand, Zod, RHF, and TailwindCSS by Vite (use some JWT :) )**.

## Features

* Add users with validation (email, password, confirm password)
* Toggle user active status
* Delete users
* Responsive and dark-themed UI

## Tech Stack

* **React + TypeScript** — frontend
* **React Hook Form + Zod** — form handling & validation
* **Zustand** — state management
* **TailwindCSS** — styling
* **Vite - build
* **JWT - some token
